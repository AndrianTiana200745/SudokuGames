# SudokuGames

Ce projet est une application de jeu de Sudoku développée sous forme de projet Java, conçue pour offrir une expérience classique du puzzle Sudoku. Il propose les éléments nécessaires pour jouer, générer ou vérifier des grilles de Sudoku.

## Structure du projet

```
SudokuGames/
├── .gitattributes
└── Sudoku/
    ├── .gitignore
    ├── Sudoku.iml
    ├── .idea/
    └── src/
```

- `.gitattributes`: Gère les configurations git pour l’uniformité entre systèmes.
- `Sudoku/`: Répertoire principal du projet Sudoku Java.
  - `.gitignore`: Exclut les fichiers/dossiers temporaires et spécifiques à l’environnement IDE.
  - `Sudoku.iml`: Fichier de configuration du module IntelliJ IDEA.
  - `.idea/`: Paramètres du projet pour IntelliJ IDEA.
  - `src/`: Code source principal de l’application.

## Fonctionnalités principales

- Interface pour jouer au Sudoku.
- Génération de nouvelles grilles de Sudoku.
- Vérification des solutions.
- Possibilité d’étendre l’application (niveau de difficulté, sauvegarde).

## Prérequis

- **Java** (v8 ou + recommandé)
- **IDE Java** tel que IntelliJ IDEA (fichiers de config inclus)

## Installation et utilisation

1. **Clonez ce dépôt :**
   ```bash
   git clone https://github.com/AndrianTiana200745/SudokuGames.git
   ```
2. **Ouvrez le projet dans IntelliJ IDEA** (recommandé pour bénéficier des fichiers de configuration inclus).
3. **Compilez le projet.**
4. **Exécutez la classe principale située dans le dossier `src/`.**

## Personnalisation

- Ajoutez vos propres grilles ou améliorez l’interface dans le dossier `src/`.
- Adaptez la logique selon le type de Sudoku (9x9, etc.).
